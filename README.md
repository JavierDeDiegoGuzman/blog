# Blog

Astro starter for:

- posts you write
- recommended reading you want to share

## Collections

- `src/content/posts/`
- `src/content/recommendations/`

## Routes

- `/` home
- `/posts` posts archive
- `/recommended-reading` recommendations archive
- `/rss.xml` posts RSS
- `/recommended-reading/rss.xml` recommendations RSS

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Notes

- Update `src/consts.ts` with your site title and description.
- Update `astro.config.mjs` with your real production `site` URL before deploying.
